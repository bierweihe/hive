    Java常用类

    # Java常用类

    # 日历类Java.util.Calendar的特点

    • #该类为抽象类，里面由许多关于日历日期的方法

    # Calendar类创建实例对象

    • 使用getInstance()方法，默认时区和区域获取当前系统事件，该方法为静态方法，
      • Calendar c = Calendar.getInstance();
    • 实例方法：
      • toString()，
      • add()
      • get(int feild)
      • getActualMaxinmum(int feild)
      • getActualMinmum(int feild)
    • 静态方法：getInstance()

    # Calendar类中add方法

    • abstract add(int feild,int amount)将指定的时间量（YEAR,MONTH,DATE）添加或减去给定的日历字段。

    # Calendar类中的get方法

    • int get(int field),通过日历对象获取当前对应时间量
    • getActualMaxinmum(int feild),获取本（月，周）第一个，getActualMinmum则相反。

    # Date是指哪个类

    • java.util.Date,日期类，主要掌握
      • 任何获得系统当前时间
      • String->Date
      • Date->String

    # java获取系统当前时间（精确到毫秒）

    • 使用构造方法Date()创建对象：
    Date nowDate = new Date();
print(nowDate);
toString已被重写，可以通过SimpleDateFormat来改变输出样式

    # SimpleDateFormat类

    • 专门负责日期格式化的类，java.text.SimpleDateFormat
    • 日期格式代表字符：
      • yyyy年，MM月，dd日，HH时，mm分，ss秒，SSS毫秒，除了以上字符不可更改其他皆可更改
    • 获取一种日期格式的对象：
    SimpleDateFormat sdf1 = new SimpleDateFormat("yyyy-MM-dd HH:mm:ss");
    • 将Date对象转换为String对象方便输出，
    • format()是将Date-->String的方法
    String nowDateStr = sdf.format(nowDate);
print(nowDateStr);
    • 将日期字符串转换为日期对象；
    String time08 = "2008年08月8日 08时08分08秒 888毫秒"
//日期格式
SimpleDateFormat sdf = new SimpleDateFormat("yyyy年MM月dd日 HH时mm分ss秒 SSS毫秒");
//用此模板来转换String为Date
Date date08 = sdf.parse(time08);
print(date08)
    • 注意：格式如果不一样会发生，解析异常

    # System类的特点

    • System类提供的设施包括标准输入，标准输出和错误输出流; 访问外部定义的属性和环境变量; 一种加载文件和库的方法; 以及用于快速复制阵列的一部分的实用方法。
    • 本身被final修饰，不能被实例化

    # System类中属性(皆为静态的)

    • static PrintStream in/out/err
    • 分别为标准输入/输出/错误流
